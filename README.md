# If-Tree Skill

If-Tree Skill 方法论的公开发布与版本记录仓库。

## 仓库用途

本仓库用于公开发布、持续版本化保存 If-Tree Skill 方法论，并作为可核验的公开记录。

建立本仓库的目的包括：

- 为该方法论提供稳定、公开、可追溯的时间戳
- 保留清晰可检查的版本演化历史
- 便于后续引用、署名与归属说明
- 减少围绕“先发表”“先公开”的争议

## 核心思想

If-Tree Skill 将条件树视为程序的自然语言形态。

其核心主张包括：

- 文档即程序，程序即文档，二者严格等价
- 条件分支与循环足以表达完整程序逻辑
- 对于用户未给出的规则，系统不得擅自猜测或补写
- 当实现所需事实不足时，文档必须显式暴露 ERROR，而不是静默补全
